## Overview

Marketing and documentation site for [persona.json](https://persona.omni.dev), the portable non-human identity specification.

## Features

- **React 19** with TanStack Start/Router
- **Tailwind CSS v4** for styling
- **Nitro v2** for SSR
- **Vite 7** for builds

## Development

```bash
bun i
bun dev
```

## Docker

```bash
docker build -t persona-json-site .
docker run -p 3000:3000 persona-json-site
```
